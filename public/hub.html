Sobre el proyecto

Contexto

Prototipo de sistema MES basado en la suite ERP de Siesa (empresa colombiana líder), diseñado para planta de autopartes. Cubre el flujo completo desde órdenes de producción hasta liberación de lotes por calidad.

Stack tecnológico

Next.js 14 TypeScript React 18 Google Sheets API Claude AI Gemini Flash Tailwind CSS Jest Vercel
Logros técnicos destacados
⚡
Arquitectura de contextos React 3 contextos globales (App, Manufactura, Calidad) sincronizados con Google Sheets en tiempo real
🔗
Flujo manufactura → calidad Al completar una operación se dispara automáticamente una inspección en calidad
🤖
IA integrada end-to-end Análisis de defectos por voz, predicción de riesgos y clasificación de NCs con Gemini AI
🔔
Sistema de notificaciones Alertas sintéticas en memoria para datos históricos sin contaminar la base de datos
📊
Trazabilidad completa de lotes Timeline de eventos con síntesis automática de historial para datos pre-existentes
✅
112 tests automatizados Suite Jest con 6 archivos cubriendo flujos de manufactura, calidad y ciclos Sheets
