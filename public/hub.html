Sobre el proyecto

Contexto

Prototipo de sistema MES basado en la suite ERP de Siesa (empresa colombiana líder en software empresarial), diseñado para plantas de manufactura de autopartes. Cubre el flujo completo desde órdenes de producción hasta liberación de lotes por calidad.

Stack tecnológico

Next.js 14 TypeScript React 18 Google Sheets API Claude AI Gemini Flash Tailwind CSS Recharts Vercel Jest

Metodología

Desarrollo 100% asistido por IA mediante prompting estructurado. Cada módulo fue definido con requerimientos funcionales precisos y construido iterativamente. Base de datos persistente en Google Sheets, desplegado en producción en Vercel.

Módulos implementados

Órdenes producción Rutas Centros de trabajo BOM / Materiales Inspecciones No conformidades Trazabilidad Muestreo Predicción IA Análisis defectos
Logros técnicos destacados
⚡
Arquitectura de contextos React 3 contextos globales (App, Manufactura, Calidad) con sincronización en tiempo real contra Google Sheets
🔗
Flujo manufactura → calidad Al completar una operación se dispara automáticamente una inspección en el módulo de calidad
🤖
IA integrada end-to-end Análisis de defectos por voz, predicción de riesgos de producción y análisis de NCs con Gemini AI
🔔
Sistema de notificaciones Alertas sintéticas en memoria para datos históricos sin contaminar la base de datos
📊
Trazabilidad completa Timeline de eventos por lote con síntesis automática de historial para datos pre-existentes
✅
112 tests automatizados Suite Jest con 6 archivos de tests cubriendo flujos de manufactura, calidad y ciclos de Sheets
